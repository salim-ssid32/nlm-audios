# nlm-audios
This repository contains a dataset of audio recordings representing the pronunciation of numbers from 0 to 10,000 in five Benin local languages: **Yoruba**, **Fongbe**, **Dendi**, **Baatonum**, and **Fufulbe**. These audio files are generated using the Facebook MMS TTS (Text-to-Speech) model.

## Overview

The dataset provides audio files for the pronunciation of numbers ranging from 0 to 10,000 across five different languages spoken in Benin. Each language has its own folder containing the corresponding audio files for each number. The data can be useful for linguistic research, speech recognition, language learning tools, and more.

### Key Features:
- **10001 audio files** for numbers 0 to 10,000.
- **5 local Benin languages**: Yoruba, Fongbe, Dendi, Baatonum, and Fufulbe.
- Audio files are generated using **Facebook's MMS TTS model** for high-quality text-to-speech synthesis.

## Languages

This dataset includes the following languages:

1. **Yoruba**  
2. **Fongbe**  
3. **Dendi**  
4. **Baatonum**  
5. **Fufulbe**

Each language has a corresponding folder containing the audio files for the numbers 0-10,000.

## Dataset Structure

The dataset is structured as follows:

```yaml
raw/
│
├── Yoruba/
│   ├── 0000.ogg
│   ├── 0001.ogg
│   ├── ...
│   └── 10000.ogg
│
├── Fongbe/
│   ├── 0000.ogg
│   ├── 0001.ogg
│   ├── ...
│   └── 10000.ogg
│
├── Dendi/
│   ├── 0000.ogg
│   ├── 0001.ogg
│   ├── ...
│   └── 10000.ogg
│
├── Baatonum/
│   ├── 0000.ogg
│   ├── 0001.ogg
│   ├── ...
│   └── 10000.ogg
│
└── Fufulbe/
    ├── 0000.ogg
    ├── 0001.ogg
    ├── ...
    └── 10000.ogg

```


- Each language folder contains audio files in `.ogg` format.
- The file names are the corresponding numbers (e.g., `0000.ogg` for "zero", `0001.ogg` for "one", and so on up to `10000.ogg`).
